Câu 1: HTML là chữ viết tắt của từ nào? (0.2 điểm)
A. Hyper Text Markup Language
B. Hyperlinks and Text Markup Language
C. Home Tool Markup Language
D. Hyper Tool Markup Language
Đáp án đúng: A

Câu 2: Thẻ nào dùng để tạo tiêu đề lớn nhất trong HTML? (0.2 điểm)
A. <heading>
B. <h6>
C. <head>
*D. <h1>

Câu 3: Để định dạng chữ in đậm trong HTML, ta dùng thẻ nào?
A. <b>
B. <important>
C. <i>
D. <bold>
Đáp án: A

Câu 4: CSS là viết tắt của từ nào? (0.4 điểm)
A. Colorful Style Sheets
B. Creative Style Sheets
C. Cascading Style Sheets
D. Computer Style Sheets
-> Đáp án: C

Câu 5: Phương thức HTTP nào thường được dùng để gửi dữ liệu lên server nhằm tạo mới một tài nguyên?
A. GET
B. POST
C. DELETE
D. PUT
Đáp án: B
